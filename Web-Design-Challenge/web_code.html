Project Explanation

The goal of this project was to examine how weather changes with lattitude. This goal was carried out by using the OpenWeatherApp API to gather data on various weather conditions in cities all over the globe. Once our data was collected, MatPlotLib was utilized to construct plots of City Lattitude vs. Humidity, Cloudiness, Max Temperature, and Wind Speed. These plots can be viewed on this website, along with a table of the data that was utilized to construct the plots and descriptions of our findings in regards to how Lattitude affects weather.
